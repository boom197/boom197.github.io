IDEA入门

发表于 2019-10-30 | 分类于 IDEA | | 次阅读
| 字数统计: 977 字 | 阅读时长 ; 3 分钟

IDEA 全称IntelliJ IDEA，是java语言开发的集成环境，IntelliJ在业界被公认为最好的java开发工具之一，尤其在智能代码助手、代码自动提示、重构、J2EE支持、Ant、JUnit、CVS整合、代码审查、 创新的GUI设计等方面的功能可以说是超常的。IDEA是JetBrains公司的产品。

创建项目

创建Java项目

打开IDEA，选择 Create New Project ，一路Next。

M：编写完java代码，怎么运行呢？

Z：点击右边的开始按钮，选择运行项目main方法

M：idea怎么进行编译呢？

Z：点击编译按钮，target中的红色文件夹中就是编译后的文件，一般运行的时候会自动编译。

创建Web项目

创建Web项目使用Java Enterprise，同时对Tomcat的路径进行配置：

选择Web Application

M：在idea中怎么创建资源文件夹？

Z：右键新建文件夹

另外还要将普通文件夹指定为资源文件夹才会被编译

创建maven的jar项目

要创建maven的jar项目，选择maven之后直接next

填入项目名和位置，点击next

创建maven的war项目

要创建maven的war项目，选择Maven，打勾通过原型创建，选择webapp

输入项目名，地址点击下一步

Z：这里要将欠缺的文件夹（java，resource）补全，并且将其设置为Sources Root和Resources Root具有功能的文件夹（转化之后才能往里面添加class和xml文件）

部署项目

M：新建完Web项目之后怎么启动呢？

Z：配置tomcat

配置tomcat

通过配置Tomcat启动，选择Edit Configurations

配置Local的Tomcat

填写tomcat名，配置tomcat位置

点击加号+，选择Artifact..

添加已解释的war包

运行tomcat即可进行访问

插件的方式启动

  1. pom.xml添加tomcat配置

2.添加Maven，填入运行指令则配置好插件tomcat

同样方式点击即可运行。


常用的快捷键

跟eclipse不同的快捷键

查找类

Ctrl+N

移动上下行

Ctrl + shift + ↑/↓

快速复制/删除行

Ctrl+D复制，Ctrl+X粘贴

搜索

Ctrl+F，搜索并替换的快捷键是Ctrl+R

项目切换

Alt+←/→ ，前进后退导航Ctrl + Alt + ←/→

全局搜索

Ctrl + N搜索文件，Ctrl + Shift + N搜索文件夹，Alt+F7(+Fn)搜索文字，Ctrl + Shift + F搜索文字并显示代码详情。

set get

Alt+Insert

重构重命名

Shift+F6 (+Fn)

重构提取方法

Ctrl+Alt+M

运行或者调试

运行Shift+F10，调试Shift+F9

断点调试

进入F7 (+Fn) ，跨出F8 (+Fn)

补全代码

main方法psvm，输出代码sout，可以写成 “输出语句的内容”.sout (string字符串+.sout)的形式。

for循环

使用arr.fori就可以直接生成一段for循环代码。foreach的循环用arr.iter

去除/导入包

Ctrl + Alt + O

新的好用的快捷键

全屏编辑模式

Ctrl+ shift + F12 (+Fn)

在类中的方法之间切换

Ctrl + F12(+Fn)

UML图

Ctrl+Alt+U

复制文件

F5(+Fn)，也有移动，是F6(+Fn)

定位错误

F2(+Fn)，上一个用Shift + F2(+Fn)

处理明显的语法错误

Ctrl + Shift + 回车

查看接口

Ctrl + Shift + B即可查看实现的类列表

参数信息

Ctrl + P可以显示参数信息进行提示

搜索所有文件

双击Shift

偶尔用到的快捷键

快速选择代码

基于语法选择使用Ctrl + W，要缩小选择范围Ctrl + Shift + W

快速格式化代码

Ctrl + Alt + L

实现方法

Ctrl + I

重写方法

Ctrl + O

查看类的继承，实现关系

Ctrl + H
